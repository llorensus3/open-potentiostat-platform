# 🧪 Chronoamperometry Laboratory Practice (Master Level)

## Overview
This laboratory session introduces chronoamperometry (CA) using the Open Potentiostat Platform.

## Objectives
- Understand current-time response
- Distinguish capacitive vs faradaic current
- Validate Cottrell behavior

## Experimental Setup
- Potentiostat (ESP32 + Raspberry Pi)
- Electrochemical cell (WE, RE, CE)

## Principle
A potential step is applied and current is measured over time.

I(t) = Ic + If

## Procedure
1. Connect electrodes
2. Apply potential step
3. Record current for 10 s

## Expected Results
- High initial current
- Decay over time

## Data Analysis
- Plot I vs t
- Plot I vs t^-1/2

## Conclusion
Chronoamperometry allows studying diffusion and electrochemical kinetics.
